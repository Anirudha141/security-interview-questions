Reconnaissance & OSINT

Tools for gathering information and mapping attack surfaces

Nmap
Network discovery and security auditing. Port scanning, service detection, OS fingerprinting, and NSE scripts.
Network
Masscan
Fastest port scanner. Can scan the entire internet in under 6 minutes. Great for large-scale reconnaissance.
Network
Amass
In-depth attack surface mapping and asset discovery. Subdomain enumeration using multiple data sources.
OSINT
Subfinder
Fast passive subdomain enumeration tool. Uses various sources like Shodan, Censys, and certificate transparency.
OSINT
Shodan
Search engine for internet-connected devices. Find exposed services, IoT devices, and vulnerable systems.
OSINT
theHarvester
Gather emails, subdomains, hosts, employee names from public sources like search engines and PGP servers.
OSINT
Recon-ng
Full-featured reconnaissance framework. Modular design similar to Metasploit for OSINT gathering.
OSINT
SpiderFoot
Automated OSINT collection tool. Gathers intelligence from 100+ data sources about IPs, domains, emails.
OSINT

Web Application Testing

Tools for testing web application security

Burp Suite
Industry-standard web security testing platform. Proxy, scanner, intruder, repeater, and extensibility via BApps.
Web
OWASP ZAP
Free and open-source web app scanner. Great alternative to Burp with active scanning and fuzzing capabilities.
Web
SQLMap
Automatic SQL injection and database takeover tool. Supports multiple database types and injection techniques.
Web
ffuf
Fast web fuzzer written in Go. Directory discovery, parameter fuzzing, virtual host discovery.
Web
Gobuster
Directory/file & DNS busting tool. Brute-force URIs, DNS subdomains, and virtual host names.
Web
Feroxbuster
Fast, recursive content discovery tool written in Rust. Auto-filters, auto-recurse, and smart wordlists.
Web
Nuclei
Fast vulnerability scanner based on templates. Huge community template library for CVEs and misconfigs.
Web
Nikto
Web server scanner. Checks for dangerous files, outdated software, and server misconfigurations.
Web
WPScan
WordPress security scanner. Enumerate users, plugins, themes, and check for known vulnerabilities.
Web
XSStrike
Advanced XSS detection suite. Fuzzing engine, context analysis, and WAF bypass capabilities.
Web
Dalfox
Fast parameter analysis and XSS scanning tool. Supports DOM XSS detection and blind XSS.
Web
httpx
Fast HTTP toolkit. Probe for working HTTP servers, extract titles, status codes, technologies.
Web

Exploitation
Cobalt Strike

Red Team
Sliver

Red Team

Red Team
Mythic

Post-Exploitation
Covenant

Active Directory & Windows

Tools for attacking and auditing Active Directory environments

BloodHound
AD attack path visualization. Maps relationships and identifies shortest paths to Domain Admin.
AD
Mimikatz
Windows credential extraction tool. Dump passwords, hashes, tickets from memory. Golden/Silver tickets.
Windows
Rubeus
Kerberos abuse toolkit. Kerberoasting, AS-REP roasting, ticket manipulation, delegation attacks.
AD
Impacket
Python library for network protocols. Includes psexec, wmiexec, secretsdump, GetUserSPNs, and more.
AD
CrackMapExec
Swiss army knife for pentesting networks. SMB, WinRM, LDAP, MSSQL enumeration and exploitation.
AD
PowerView
PowerShell AD enumeration tool. Part of PowerSploit. Domain enumeration, ACL analysis, trust mapping.
AD
Certify
Active Directory Certificate Services (ADCS) enumeration and abuse tool. Find and exploit ESC vulnerabilities.
AD
Certipy
Python tool for ADCS attacks. Enumerate, request, and forge certificates for domain takeover.
AD
SharpHound
BloodHound data collector. Gathers AD data including users, groups, sessions, ACLs for analysis.
AD
ADRecon
Active Directory reconnaissance tool. Generates comprehensive Excel report of AD environment.
AD
Responder
LLMNR, NBT-NS, MDNS poisoner. Captures NTLMv1/v2 hashes on the network for cracking or relay.
Network
PetitPotam
Coerce Windows hosts to authenticate via MS-EFSRPC. Used for NTLM relay attacks.
AD

Password Attacks

Tools for cracking and attacking passwords

Hashcat
World's fastest password cracker. GPU-accelerated, supports 300+ hash types, multiple attack modes.
Cracking
John the Ripper
Versatile password cracker. Auto-detects hash types, supports many formats, rule-based attacks.
Cracking
Hydra
Online password brute-forcer. Supports 50+ protocols including SSH, FTP, HTTP, SMB, RDP.
Brute Force
CeWL
Custom wordlist generator. Spiders websites to create targeted wordlists for password attacks.
Wordlist
Spray
Password spraying tool. Test single passwords against many users to avoid lockouts.
Brute Force

Analysis

MITMf

hping3
